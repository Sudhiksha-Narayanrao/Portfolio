# Sudhiksha Narayanraopeta - Portfolio

A custom, interactive portfolio designed with a retro aesthetic. This project showcases my journey as a Computer Science student, focusing on AI, Machine Learning, and Spatial Data Science.

## Live Demo
**[CLICK HERE TO VIEW MY PORTFOLIO](https://Sudhiksha-Narayanrao.github.io/Portfolio)** 

## Features
- **Retro OS Interface:** Navigation inspired by classic desktop environments.
- **Interactive Folders:** Explore my education, certifications, and hobbies through popup windows.
- **Project Sidebar:** Dedicated section for my work in DNN and Image Processing.
- **Responsive Design:** Built using React.js and custom CSS.

## Tech Stack
- **Frontend:** React.js, CSS3
- **Tools:** GitHub Pages, VS Code
- **Languages Featured:** JavaScript

## Project Structure
- `src/Home.js`: The main logic for the desktop environment and folder contents.
- `src/App.css`: Custom styling for the grid, taskbar, and retro windows.
- `src/photo.jpg`: Profile imagery used in the intro card.
